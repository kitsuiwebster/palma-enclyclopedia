APIs

This application uses the following APIs to provide palm data and related services.

Available APIs

  • GET /api/palms - Retrieve a list of all palm species.
  • GET /api/palms/{{ '{' }}id{{ '}' }} - Get detailed information about a specific palm.
  • GET /api/traits - Fetch available palm traits and characteristics.
  • GET /api/photos - Access palm images and galleries.

API Usage

Use these endpoints to integrate palm data into your applications or research projects.
